Topic Extraction

The user goes to the homepage and selects Explore.

The user chooses the required feature but must be registered and logged in to proceed.

The user selects the file type to upload.

The user uploads files or proceeds to file preview and selection.

The user views the uploaded files and selects the required ones.

The user selects the file and is shown a preview of the data as well as the topics selection button.

The user selects the no of Topics they would like to extract from the corpus.

When the processing is finished the user selects the view results

Above we have the results of a topic extraction on a 100 document corpus.

  • The results in the first panel show the distribution of documents to topic results in bar
    chart form.
  • The second panel shows the distribution in a pie chart format.
  • The third panel shows the first 15 documents in the file, the first 10 words, and the topic
    the document was assigned.
  • The fourth panel shows each of the topics and the word distribution that was extracted to make
    up that topic.

The user then has the option of downloading the results in a CSV file or saving a copy to storage

Document Extraction

The user goes to the homepage and selects Explore.

The user chooses the required feature but must be registered and logged in to proceed.

The user selects the file type to upload.

The user uploads files or proceeds to file preview and selection.

The user views the uploaded files and selects the required ones.

The user selects extract on the image file that they want to extract text from. The page will
then show a processing screen until the data is returned from textract.

After processing a preview of the extracted data as well as all previously extracted files and
any CSV files available the user receives a message that the text was extracted successfully and
the text has been added to a file and uploaded to the user's storage.

The user can then either add the extracted text to a CSV file or return to the file page to
extract from a new image file.

After adding the text to the CSV file the user is redirected to the feature selection page
with a successful confirmation message


Summarisation

The user goes to the homepage and selects Explore.

The user chooses the required feature but must be registered and logged in to proceed.

The user selects the file type to upload.

The user uploads files or proceeds to file preview and selection.

The user views the uploaded files and selects the required ones.

The user selects the file and is shown a preview of the data as well as all CSV files available

The user selects the begin summarisation.

The user waits for the summarization process to finish then selects view results

Above we have the results of document summary on a 100 document corpus.

  • The results in the first panel show the summarization results in bar chart form.
  • The second panel shows the distribution in a pie chart format
  • The third panel shows the first 10 documents in the file, the first 15 words of each original
    file, the character length, t the first 15 words of the summarization, and the new
    summarised character length.

The user then has the option of downloading the results in a CSV file or saving a copy to storage
